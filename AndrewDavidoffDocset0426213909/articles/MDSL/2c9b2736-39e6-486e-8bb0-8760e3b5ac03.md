 <b>PopupAnimationSyntax</b> </td>
 </tr>
 <tr><td><div class="indent0">[values]</div></td>
 <td></td>
 </tr>
 <tr><td><div class="indent2">Fade</div></td>
 <td>The Popup control gradually appears, or fades in. This effect is created by increasing the opacity of the Popup window over time.</td>
 </tr>
 <tr><td><div class="indent2">None</div></td>
 <td>The Popup control appears without animation.</td>
 </tr>
 <tr><td><div class="indent2">Scroll</div></td>
 <td>The Popup control scrolls from the upper-left corner of its parent. If the screen does not provide enough room to allow the Popup default behavior, the Popup scrolls from the lower-right corner instead.</td>
 </tr>
 <tr><td><div class="indent2">Slide</div></td>
 <td>The Popup control slides down or up into place. By default, a Popup slides down. However, if the screen does not provide enough room for the Popup to slide down, it slides up instead.</td>
 </tr>
