					<p xmlns="">An XML document is a Xaml Document if a well-formed Xaml Information Set can be created from the XML document’s XML Infoset. The conversion process is performed with reference to a set of schemas that includes those defined in Section 5, “Intrinsic Schema Information Items”. Additional vocabulary-specific schemas may (and usually will) also be used as part of the conversion process.</p>
					<p xmlns="">In order to construct a Xaml Information set from an XML document, that document’s XML Infoset is processed with the rules defined in section 6.6. If the rules in section 6.6 are successfully executed without error, the resulting Xaml Information Set must conform to the constraints in Section 4 in order for the XML document to be said to be a Xaml Document.</p>
