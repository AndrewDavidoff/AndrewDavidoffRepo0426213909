TileModeSyntax </b>
 </td>
 </tr>
 <tr><td><div class="indent0">[values]</div></td>
 <td> </td>
 </tr>
 <tr><td><div class="indent2">FlipX</div></td>
 <td>The same as Tile except that alternate columns of tiles are flipped horizontally. The base tile itself is not flipped. </td>
 </tr>
 <tr><td><div class="indent2">FlipXY</div></td>
 <td>The combination of FlipX and FlipY. The base tile itself is not flipped. </td>
 </tr>
 <tr><td><div class="indent2">FlipY</div></td>
 <td>The same as Tile except that alternate rows of tiles are flipped vertically. The base tile itself is not flipped. </td>
 </tr>
 <tr><td><div class="indent2">None</div></td>
 <td>The base tile is drawn but not repeated. The remaining area is transparent </td>
 </tr>
 <tr><td><div class="indent2">Tile</div></td>
 <td>The base tile is drawn and the remaining area is filled by repeating the base tile. The right edge of one tile meets the left edge of the next, and similarly for the bottom and top edges. </td>
 </tr>
