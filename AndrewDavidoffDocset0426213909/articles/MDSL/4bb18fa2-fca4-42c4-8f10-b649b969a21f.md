 <b>ScrollBarVisibilitySyntax</b> </td>
 </tr>
 <tr><td><div class="indent0">[values]</div></td>
 <td></td>
 </tr>
 <tr><td><div class="indent2">Auto</div></td>
 <td>A ScrollBar appears and the dimension of the ScrollViewer is applied to the content when the viewport cannot display all of the content. For a horizontal ScrollBar, the width of the content is set to the ViewportWidth of the ScrollViewer. For a vertical ScrollBar, the height of the content is set to the ViewportHeight of the ScrollViewer.</td>
 </tr>
 <tr><td><div class="indent2">Disabled</div></td>
 <td>A ScrollBar does not appear even when the viewport cannot display all of the content. The dimension of the content is set to the corresponding dimension of the ScrollViewer parent. For a horizontal ScrollBar, the width of the content is set to the ViewportWidth of the ScrollViewer. For a vertical ScrollBar, the height of the content is set to the ViewportHeight of the ScrollViewer.</td>
 </tr>
 <tr><td><div class="indent2">Hidden</div></td>
 <td>A ScrollBar does not appear even when the viewport cannot display all of the content. The dimension of the ScrollViewer is not applied to the content.</td>
 </tr>
 <tr><td><div class="indent2">Visible</div></td>
 <td>A ScrollBar always appears. The dimension of the ScrollViewer is applied to the content. For a horizontal ScrollBar, the width of the content is set to the ViewportWidth of the ScrollViewer. For a vertical ScrollBar, the height of the content is set to the ViewportHeight of the ScrollViewer.</td>
 </tr>
