BitmapCacheOptionSyntax </b>
 </td>
 </tr>
 <tr><td><div class="indent0">[values]</div></td>
 <td> </td>
 </tr>
 <tr><td><div class="indent2">Default</div></td>
 <td>Caches the entire image into memory. This is the default value. </td>
 </tr>
 <tr><td><div class="indent2">None</div></td>
 <td>Do not create a memory store. All requests for the image are filled directly by the image file. </td>
 </tr>
 <tr><td><div class="indent2">OnDemand</div></td>
 <td>Creates a memory store for requested data only. The first request loads the image directly; subsequent requests are filled from the cache. </td>
 </tr>
 <tr><td><div class="indent2">OnLoad</div></td>
 <td>Caches the entire image into memory at load time. All requests for image data are filled from the memory store. </td>
 </tr>
