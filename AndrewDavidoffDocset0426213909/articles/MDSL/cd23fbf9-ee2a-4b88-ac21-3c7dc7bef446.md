						<b>About This Specification</b>
					</p>
					<p xmlns="">Xaml, the eXtensible Application Markup Language, is a system for representing structured information. This specification defines three aspects of Xaml:</p>
					<ul xmlns=""><li class="unordered">
							<p class="BulletedList">The Xaml Schema Information Set – a model for defining a particular Xaml vocabulary.</p>
						</li><li class="unordered">
							<p class="BulletedList">The Xaml Information Set – a model for describing the information in a Xaml instance.</p>
						</li><li class="unordered">
							<p class="BulletedList">The process for converting an XML [XML] document into the corresponding Xaml Information Set, as directed by one or more Xaml Schema Information Sets.</p>
						</li></ul>
					<p xmlns="">This specification does not define any particular Xaml vocabulary.</p>
