 <h1 class="heading">2.2.4.299 IPBlockRootEnumerationParameters</h1>
 </div>
 <div id="mainSection">
 <div id="mainBody">
 <div id="allHistory" class="saveHistory"></div>
 <div id="sectionSection0" class="section" name="collapseableSection">
 

<p>The IPBlockRootEnumerationParameters complex type is used to
specify the criteria for enumerating the rows that form the first level of
address blocks (for which there are no parent blocks).</p>

<dl>
<dd>
<div><pre> &lt;xs:complexType name=&quot;IPBlockRootEnumerationParameters&quot;&gt;
   &lt;xs:complexContent mixed=&quot;false&quot;&gt;
     &lt;xs:extension base=&quot;ipam:EnumerationParametersBase&quot;&gt;
       &lt;xs:sequence&gt;
         &lt;xs:element minOccurs=&quot;0&quot; name=&quot;AddressCategory&quot; nillable=&quot;true&quot; type=&quot;ipam:AddressCategory&quot; /&gt;
         &lt;xs:element minOccurs=&quot;0&quot; name=&quot;AddressFamily&quot; type=&quot;syssock:AddressFamily&quot; /&gt;
       &lt;/xs:sequence&gt;
     &lt;/xs:extension&gt;
   &lt;/xs:complexContent&gt;
 &lt;/xs:complexType&gt;
</pre></div>
</dd></dl>

<p><b>AddressFamily</b>: This specifies the address
family of the address block instances that need to be enumerated.</p>

<p><b>AddressCategory</b>: This specifies the address
category of the address blocks that need to be enumerated.</p>
