<p>The IpamNumberOfRowsObject complex type defines the type
that can be used to return an integer value specifying the number of rows as a
result from the IPAM enumeration port types.</p>

 &lt;/xs:complexType&gt;
</pre></div>

specifying the number of rows returned.</p>
